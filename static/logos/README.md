## 如何添加更多Logo

### 方法1: 从官方网站下载
1. 访问公司/大学的官方网站
2. 查找"媒体资源"、"品牌指南"或"Press Kit"页面
3. 下载官方logo（PNG格式，透明背景，建议尺寸200x200px）
4. 将文件保存到此目录，使用以下命名规则：
   - 科技公司：使用公司英文名（小写，连字符分隔），如 `tesla.png`, `nvidia.png`
   - 金融机构：使用公司英文名，如 `morgan-stanley.png`, `jpmorgan.png`
   - 大学：使用缩写，如 `uq.png`, `oxford.png`, `berklee.png`

### 方法2: 从Wikipedia Commons下载
访问 https://commons.wikimedia.org 搜索公司/大学名称，下载SVG或PNG格式的logo。

### 方法3: 使用品牌资源库
- logos-world.net
- brandfetch.io
- 各公司的官方媒体资源页面

## 文件要求
- 格式：PNG（推荐）或SVG
- 背景：透明
- 尺寸：建议200x200px或更大
- 文件大小：建议小于500KB
